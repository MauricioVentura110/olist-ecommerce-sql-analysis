# E-Commerce Data Analysis with SQL (Olist Dataset)

## Project Overview

This project performs a business-oriented analysis of an e-commerce dataset using SQL.  
The goal is to simulate the type of analytical work performed by data analysts in real business environments.

The project focuses on answering practical business questions related to:

- sales performance
- customer behavior
- seller performance
- delivery efficiency
- marketplace growth trends

The analysis is conducted using **PostgreSQL** and organized as a reproducible SQL project.

---

## Dataset

This project uses the **Brazilian E-Commerce Public Dataset by Olist**, which contains real transactional data from a Brazilian online marketplace.

The dataset includes information about:

- customers
- orders
- order items
- payments
- sellers
- products
- reviews

Dataset source:

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

---

## Tools Used

- PostgreSQL (database system)

- GitHub (project version control)

## Database Structure

The database used in this project consists of **7 tables** derived from the original dataset.

Main entities:

- customers
- orders
- order_items
- order_payments
- products
- sellers
- order_reviews

These tables allow reconstruction of the full e-commerce workflow:

Customer → Order → Order Items → Payment → Delivery → Review

---

## Data Cleaning

During the import process, the table **`order_reviews`** contained inconsistent formatting that prevented it from being imported correctly into the database through DBeaver.

To solve this issue, the dataset was cleaned using **Python** before loading it into the database.

The cleaning script can be found in:

```
data_cleaning/clean_order_reviews.py
```

This preprocessing step ensures that the dataset can be imported properly and analyzed using SQL.

## Business Questions Addressed

The SQL analysis answers several questions that are common in real business analytics environments.

Examples include:

### Sales Analysis

- What is the total revenue generated by the platform?
- How have sales evolved over time (by month)?
- What are the monthly sales trends?

### Customer Analysis

- Who are the top customers by total spending?
- How many customers are repeat buyers?
- What is the average order value?

### Seller Performance

- Which sellers generate the highest revenue?
- Which sellers sell the largest number of products?

### Delivery Performance

- What is the average delivery time?
- What percentage of orders are delivered after the estimated delivery date?
